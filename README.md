# Mango-Pest-Detect-By-Nattapong-Saithin
🍋 Mango Pest Classification using ML/AI
This repository contains my first project in Machine Learning and Artificial Intelligence, focusing on the classification of mango pests.
The goal is to build a model that can automatically identify and classify different types of pests affecting mango crops, helping farmers and researchers improve pest management and crop yield.

🌱 Motivation
Mango is one of the most important fruits in tropical regions, but pests significantly reduce yield and quality.
By applying AI-based classification, this project aims to support smart agriculture and contribute to sustainable farming practices.
